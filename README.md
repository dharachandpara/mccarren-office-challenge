# McCarren Office Challenge (Word Add-in)

A simple Microsoft Word taskpane add-in that helps create quick notes.

## Features
- **Insert Template**: inserts a prebuilt summary template into the document.
- **Bullets from Selection**: turns selected text into a bulleted list.
  - Works with:
    - one item per line
    - `apple\banana\cherry`
    - `apple, banana, cherry`

## Run locally
```bash
npm install
npm start
